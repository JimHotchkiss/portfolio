I am at my Best, when I'm creating.

LeadTracker Desktop Application

This project was inspired by a situation I noticed at work: How do our sales representatives keep track of their leads. The user can create, read, update and destroy leads. Additionally, there is authentication, with a user being able to create an account and login. This is built with Rails api, for the backend, React running the frontend and Redux handling state management. I utilized the react-chartjs-2 api to most efficiently display the user's data.
Video Demo
Github: Lead Tracker Frontend Github: Lead Tracker API

The VeganCow Recipe App

This project was motivated by my wife needing an easy way to save and organize her vegan recipes. This project allows the user to create and store a recipe. The user can view all the available recipes, or search by category. This project was built with Ruby on Rails and Javascript managing the UI.
Video Demo
Github: The Vegan Cow

The Project Manager

This project allows the user to create software projects, keep track of of tasks, and update the progress of the project. This Rails application leveraged Active Record associations to build a powerful and scalable application, applied MVC best practices to assure a robust development environment and performance and deployed Omniauth to handle user authentication safely through Google.
Video Demo
Github: The Project Manager

Tech Support Application

This project solved a problem for our sales force team. When setting up an OR for surgery, our sales reps needed access to the best settings for each combination of visual equipment and surgical proceedures. This app gives the reps this information in an easy to access and easy to view format, making their fastpace workflows a little more managable. This project was built with html, CSS and javascript. This project earned me an Inventation Certificate from my employer.
