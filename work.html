LeadTracker Desktop Application

This project was motivated by a situation I noticed at work: How do our sales representatives keep track of their leads. The user can create, read, unpdate and destroy leads. Additionally, there is authentication, with a user able to create an account and login. This is built with Rails api, for the backen, and React running the frontend. I utilized the react-chartjs-2 api to most efficiently display the user's data.
https://github.com/JimHotchkiss/lead-tracker-client https://github.com/JimHotchkiss/lead-tracker-api

The VeganCow Recipe App

This project was motivated by my wife needing an easy way to save and organize her vegan recipes. This project allows the user to create and store a recipe. The user can view all the available recipes, or search by category. This project was built with Ruby on Rails and Javascript managing the UI.
https://github.com/JimHotchkiss/The_Vegan_Cow

Tech Support Application

This project solved a problem for our sales force team. When setting up an OR for surgery, our sales reps needed access to the best settings for each combination of visual equipment and surgical proceedures. This app gives the reps this information in an easy to access and easy to view format, making their fastpace workflows a little more managable. This project was built with html, CSS and javascript.
